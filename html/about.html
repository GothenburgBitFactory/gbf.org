GothenburgBitFactory

GothenburgBitFactory is a collection of technologies and products that provide support for the activities surrounding the development, testing and distribution of Taskwarrior and Timewarrior

There are many projects developed by gothenburgbitfactory.org.

 

The technologies currently in use are:
  • Git for source management
  • Gitea for git hosting and repository management
  • Atlassian Jira for issue tracking
  • Caddy Server for HTTP(S) server
  • Python for unit testing
Services generously provided by:
  • RackSpace hosting
  • Digital Ocean hosting
  • Pingdom monitoring
  • StatusPage monitoring
  • DNSimple DNS hosting
